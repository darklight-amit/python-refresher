Git Commands

Command description
git --version show the currenlty installed version
git init initialize the git repository
git status current state of repo
git add To stage the files from local which are ready to commit and track these changes
git commit -m "message" Commit the tracked changes
git log show the full git log
git branch {branch name} Create new branch from Master
git branch Display all branches and current one with * sign
git checkout {branch name} Switched to the branch
